# Authentication

For more information, check out the [documentation](../../docs/ctx.md).

In one terminal, start the server.

```sh
cargo run
```

In another terminal, install TypeScript dependencies, and run the client!

```sh
cd auth-demo
corepack enable # if not enabled
pnpm i
pnpm dev
```
